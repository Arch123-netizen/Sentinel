# Sentinel

> AI-assisted phishing detection browser extension.

## Overview 

Sentinel helps users identify phishing websites by combining rule-based analysis with AI-generated explanations.

Instead of simply displaying a warning, Sentinel explains why a website may be dangerous, helping users recognize phishing techniques and improve their cybersecurity awareness.
