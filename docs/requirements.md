Who is Sentinel built for?
- Sentinel is built for everyday users, especially those with less cybersecurity knowledge to avoid a possible phising attack and stay secure while using an online platform.

What exact problem does it solve?
- Sentinel analyzes a website to check for suspicious charecterisitics before entering personal information so users can make safer decisions.

How is it different form Chrome's built-in phising protection?
- Sentinel focuses on helping users understand why a website appears suspicous by analyzing the websites charecterisitcs and giving clear, understandable explanations about the website. It emphasizes teaching rather than replacing the in-built protections of the browser.

How would Version 1.0 look like?
- Version 1.0 must be capable of analyzing website URLs, provides understandable AI-assisted explanations and generates a risk score to check whether or not the website is safe.
